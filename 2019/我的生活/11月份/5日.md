# 2019.11.5

## 数据结构也慢慢开始吧

因为Vue项目前期的准备工作比较繁琐，什么素材模板还有各种类库。而且后台的代码也得自己写，这是比较麻烦的。感觉好麻烦，所以今天就先简单看了看数据结构，毕竟是学过一点点的，所以刚开始的时候也比较容易。因为JavaScript本身是没有指针的，所以什么数据结构都要自己去定义一个类或者对象。其实想明白了也是蛮简单的，我主要就是想学习一些算法。去年学数据结构的时候太痛苦了，当时也是刚刚接触C，C都没怎么学会，数据结构简直就是天书奇谈。现在回想起来，其实当时也是学到了不少的知识，现在理解起来就会简单很多。
